NAME
  energymonitor - monitors the status of an induction matrix connected to this computer

SYNOPSIS
  energymonitor [ADDRESS]

DESCRIPTION
  energymonitor is a full-screen GUI application for monitoring the input, output and energy levels of an induction matrix connected to this computer.

OPTIONS
  ADDRESS [optional] - specify an optional address to monitor, useful if you have more than 1 induction matrix connected to the same computer.
